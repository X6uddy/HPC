# Удаление шума Salt-and-Pepper (CPU vs GPU)

## Обзор

В данной работе исследуется разница в производительности между CPU и GPU при реализации алгоритма удаления шума типа Salt-and-Pepper для изображений в оттенках серого.

Цель эксперимента — оценить прирост производительности при использовании графического процессора в задаче, допускающей высокий уровень параллелизма.

---

## Описание алгоритма

Шум Salt-and-Pepper проявляется в виде случайных чёрных (0) и белых (255) пикселей, искажающих изображение.

Для его устранения используется медианный фильтр:

- каждый пиксель обрабатывается независимо  
- используется локальное окно 3×3  
- центральное значение заменяется медианой соседних значений  

Такой метод эффективно удаляет выбросы, сохраняя границы объектов.

---

## Реализация

Реализованы две версии алгоритма:

- **CPU** — последовательная обработка с использованием NumPy  
- **GPU** — параллельная реализация с использованием CUDA (Numba)  

GPU-версия обрабатывает множество пикселей одновременно, распределяя вычисления между потоками.

---

## Аппаратная конфигурация

- CPU: Intel Xeon E5  
- GPU: NVIDIA T4

---

## Анализ

GPU показывает ускорение примерно в **360 раз** по сравнению с CPU.

Причины высокой производительности:

- параллельная обработка большого числа пикселей  
- независимость вычислений  
- одинаковый тип операций для всех элементов  

Ограничения ускорения:

- накладные расходы на передачу данных CPU ↔ GPU  
- время запуска CUDA-ядер  
- размер изображения  

---

## Конвейер обработки

1. Загрузка изображения в градациях серого  
2. Добавление шума Salt-and-Pepper  
3. Применение медианного фильтра:
   - CPU версия  
   - GPU версия  
4. Сравнение результатов  
5. Замер времени выполнения
